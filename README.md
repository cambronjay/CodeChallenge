# Code Challenge

To use this app simply follow these steps:

1. Install latest Node.js LTS
2. npm install -g ionic
3. cd ./CodeChallenge
4. run npm install
4. ionic serve

To see the answer to Question 8 follow these steps:

1. Install latest Node.js LTS
3. cd ./node-server
2. run npm install
3. npm start
**View the results in the conole or in the terminal
